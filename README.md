# Умные заметки с ИИ

Сервис заметок на основе ИИ, созданный с использованием микросервисной архитектуры и следующих технологий:

- Django (основной бэкенд)
- FastAPI (микросервис с искусственным интеллектом)
- YandexGPT (интеграция с большими языковыми моделями)

------------------------------------------------------------------------

## Архитектура

Клиент → Django (бэкенд) → FastAPI (AI микросервис) → YandexGPT

-   **Django** отвечает за работу с пользователями, заметками, аутентификацией и бизнес-логикой.
-   **FastAPI** отвечает за обработку данных с помощью ИИ и взаимодействие с YandexGPT.
-   **YandexGPT** предоставляет возможности обобщения текста.

------------------------------------------------------------------------

## Особенности

- Создание заметок и управление ими
- Обобщение текста с помощью искусственного интеллекта
- Архитектура бэкенда на основе микросервисов
- Конфигурация на основе среды
- Четкое разделение бизнес-логики и логики искусственного интеллекта

## Локальный запуск

### 1️⃣ Django Backend

    cd backend
    pip install -r requirements.txt
    python manage.py migrate
    python manage.py runserver

Runs on: http://127.0.0.1:8000

------------------------------------------------------------------------

### 2️⃣ FastAPI AI Service

    cd ai_service
    pip install -r requirements.txt
    uvicorn app.main:app --reload --port 8001

Runs on: http://127.0.0.1:8001

------------------------------------------------------------------------

## Пример использования в Postman

### AI-функции

POST http://127.0.0.1:8000/api/ai/summarize/

Request body:

``` json
{
  "text": "Добавьте свою заметку"
}
```

Response:

``` json
{
  "summary": "Краткое изложение основных мыслей"
}
```
